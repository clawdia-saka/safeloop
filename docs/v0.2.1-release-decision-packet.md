# SafeLoop v0.2.1 Release Decision Packet

## Product positioning

SafeLoop = agent action safety evidence layer.

SafeLoop records what an agent did, what can be exactly rolled back locally, what external actions require compensation or manual review, and what the operator should decide next.

日本語: SafeLoop は、AIエージェントの行動証跡レイヤー。何をしたか、何がローカルで正確に戻せるか、何が外部副作用として補償・手動確認対象か、operator が次に何を判断すべきかを packet として残す。

## Boundary

- exact rollback only for covered local file changes
- external actions exact_rollback=false
- compensation/manual-review only
- no hosted control plane
- no remote transparency log
- local artifacts are tamper-evident review aids, not tamper-proof guarantees
