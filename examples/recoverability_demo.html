SafeLoop 0.1.4 · recoverability-first runtime

Know what changed. Roll back covered local files. Never pretend external effects vanished.

SafeLoop's first-run story is intentionally narrow: wrap a local agent run, inspect the timeline, verify tamper-evident artifacts, review a rollback plan, then apply rollback for covered local file changes.

1. Watchsafeloop watch-run
2. Inspectsafeloop timeline
3. Verifysafeloop verify-artifacts
4. Plansafeloop rollback plan
5. Applysafeloop rollback apply
